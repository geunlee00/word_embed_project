# 깃허브와 파이참 프로젝트 연결 (마스터)

1. 깃허브에 리포지토리(저장소) 만들기
    └─ URL 복사

2. 파이참 실행 > 새 프로젝트 만들기 (리포지토리 이름과 프로젝트명 동일하게)

3. 깃허브 저장소와 파이참 프로젝트 연결
    ├─ git remote add origin https://github.com/geunlee00/word_embed_project.git
    └─ 연결 확인 : git remote -v

4. 마스터가 프로젝트 구조에 대한 설정과 패키지 지정 등 파일 작업 진행
# .gitignore 파일 작성 (깃허브에 푸시되면 안되는 파일 및 폴더를 설정)
